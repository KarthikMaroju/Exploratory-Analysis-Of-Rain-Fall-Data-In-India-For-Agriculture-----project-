# Exploratory Analysis of Rainfall Data in India for Agriculture

## Project Description
This project analyzes rainfall data and predicts rainfall using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Flask
- HTML & CSS

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run model building:
   python model_building.py

3. Run application:
   python app.py

4. Open browser:
   http://127.0.0.1:5000/
